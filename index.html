O Som das Cores é um sistema interativo que tem como base a associação entre cor e som. O objetivo é que cada utilizador faça um desenho livre, à sua escolha, para que, no fim, possa ouvir como as cores usadas foram transformadas em som.
 
INSTRUÇÕES
1. Escolher uma cor de fundo, usando as letras do teclado.
2. Desenhar sobre a tela, selecionando a cor de caneta pretendida, na 2ª coluna de cores.
3. Depois do desenho terminado, clicar no botão de guardar.
4. Por fim, premir o botão play para ouvir o resultado final.
 
IMPORTANTE:
- Para a cor de fundo funcionar corretamente, o CAPSLOCK tem de estar ligado.
- Para apagar, usar o botão da borracha.
- Para mudar o tamanho da caneta, usar o slider.
- Para recomeçar, clicar no botão de reset (cruz).
- Este projeto é também adequado para crianças. Para ativar a versão infantil, basta premir o botão com o icone da criança.
